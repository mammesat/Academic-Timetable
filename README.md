# Academic Timetable 2018

A professional, high-performance web application designed for **Nafyad School** students to view their class schedules (Grades 11 & 12) with ease.

## ✨ Features
- **Dynamic Filtering:** Instantly switch between grades and sections.
- **Daily Inspiration:** Powered by Google Gemini to provide a fresh motivational quote every day.
- **Mobile Responsive:** Works beautifully on Phones, Tablets, and Desktops.
- **Professional Documentation:** Includes a built-in guide explaining the technical architecture.
- **Developer Profile:** Integrated portfolio for the developer.

## 🚀 Deployment
This project is configured for cPanel Git Deployment using `.cpanel.yml`. 

## 💻 Tech Stack
- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Performance:** Static JSON data structure for lightning-fast loads.
- **AI Integration:** Google Generative AI (Gemini API).
